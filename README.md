# BulkTrack v1

Discipline tracking web app.

## Features
- PIN Login system
- First-run onboarding (height / weight / goal)
- Dashboard daily score
- Workout tracker (weekly streak logic)
- Weight tracker (trend system)
- Habit tracker (7-day view)
- LocalStorage data persistence

## Stack
- React
- Vite
- React Router
- TailwindCSS

Currently building and iterating toward public v1 release.
